송장 변환기 v0.2 (골격 — 날짜 보존·코드표 오인 방지)

채널 주문 파일 → 이름 변환(마스터 상품명 형식) + 공란 처리 → e-포스트용 .xlsx — 설계도 v2 §3

1) 마스터 매핑 불러오기 (하루 1번이면 충분)

아직 안 불러옴 — URL·토큰 입력 후 [매핑 불러오기]

2) 채널 주문 파일 넣기 (여러 개 한꺼번에 가능)

파일을 여기로 끌어다 놓거나 클릭해서 선택 (xlsx · xls · csv · html)
암호 걸린 파일은 엑셀에서 암호 해제 후 저장해서 넣기
사용법 · 채널별 규칙 (펼치기)
  • 흐름: 매핑 불러오기 → 파일 끌어넣기 → 채널 자동 식별 확인 → [변환 · 다운로드] → 받은 파일을 기존 e-포스트 양식으로 업로드 (업로드 절차 무변경)
  • 스스: 첫 행(안내문) 자동 삭제 · D(상품명) ← T(옵션관리코드) 매핑 · E(옵션정보) 공란
  • 쿠팡(판배): K(등록상품명) ← O(옵션ID) 매핑 · L·M·P 공란
  • 선물하기: G(상품명) ← F(채널상품번호)+H(옵션) 매핑 · H 공란 · F 숫자화 — 톡스토어는 "미매핑"으로 뜸 → 현행대로 수기. ★ 주문 파일을 넣어야 함 (코드표 파일 아님)
  • 카페24·우체국쇼핑·SSG: 준비돼 있으나 미검증 — 3채널 연습 통과 후 활성 검증
  • 미매핑 행: 원본 그대로 두고 목록으로 표시 → 마스터 탭에 기입 보완 후 다시 변환하면 됨(재변환 안전)
  • 채널 식별이 틀리면 파일 카드의 선택 상자에서 직접 고르면 됨. [헤더 보기]는 식별 실패 시 캡처 협업용.
